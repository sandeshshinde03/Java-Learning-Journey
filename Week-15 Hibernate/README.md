# Hibernate – Basics, Annotations & CRUD Practice

This repository demonstrates the fundamentals of **Hibernate** in Java.  
It covers **Hibernate configuration, core annotations, CRUD operations, and entity relationship mappings** using annotation-based mapping.

---

## 📌 What is Hibernate?

Hibernate is an **Object-Relational Mapping (ORM)** framework that allows Java applications to interact with relational databases using **Java objects instead of SQL queries**.

### Benefits of Hibernate

- Reduces boilerplate JDBC code
- Database-independent
- Automatic table creation
- Transaction management
- Caching support
- Object-oriented data handling

---

## 🧩 Core Hibernate Components

| Component      | Description                                      |
| -------------- | ------------------------------------------------ |
| SessionFactory | Heavyweight object, created once per application |
| Session        | Used to perform CRUD operations                  |
| Transaction    | Manages atomic operations                        |
| Configuration  | Loads Hibernate settings                         |
